Gain a deeper understanding of ASP.NET Core by exploring its source code    |    Thien Nguyen
By exploring the source code of asp.net core, you can uncover its architecture principles and design patterns that underpin its functionality. In this session, I will work you through important parts of asp.net core source code like routing, how aspnet core process requests, how authentication, authorization work under the hood.
10:00 AM - 10:45 AM
Simplify the development of distributed applications with .NET Aspire 9    |    Thang Chung
This presentation will introduce the new features in .NET Aspire 9 and demonstrate how they can simplify the development and testing of distributed applications. By the end of the presentation, we will showcase distributed apps within the coffee shop business domain to illustrate all the capabilities of .NET Aspire 9.

11:00 AM - 11:45 AM
Boosting Single Page Application Security with Backend for Frontend Pattern and YARP    |    Phong Nguyen
In this presentation, we will explore how to secure access tokens for single page applications (SPAs) using the Backend for Frontend (BFF) pattern in conjunction with YARP (Yet Another Reverse Proxy). We will discuss the architecture of BFF, its benefits in enhancing security, and how YARP can streamline token management and API interactions.
11:50 AM - 12:35 AM
Build multi-agent GenAI apps with AutoGen    |    Phi Huynh
AutoGen is an innovative framework from Microsoft Research designed to streamline the creation of multi-agent GenAI applications. Agents in AutoGen can communicate seamlessly, coordinate with each other, or even include humans in the loop to handle tasks ranging from simple to complex through natural conversations. With support for models like OpenAI, Mistral, Gemini, and Ollama, AutoGen simplifies integration efforts, allowing developers to focus on solving real-world problems.
Let's dive into this powerful framework and discover how we can build multi-agent GenAI apps with AutoGen.
